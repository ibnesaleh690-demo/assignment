what is the difference between Hex and RGB Values?

RGB

RGB stands for Red, Green, Blue — the three primary colors of light.
Each color is made by mixing these three in different intensities.
The value of each color channel ranges from 0 to 255:
0 means none of that color.
255 means the maximum intensity.

HEX

It’s another way to describe the same RGB color, but in base-16 (hexadecimal) numbers instead of decimal (base-10).
The Hex code is divided into three pairs: #RRGGBB
Each pair (RR, GG, BB) represents the red, green, and blue intensity from 00 to FF.
00 = 0 intensity, FF = 255 intensity.
